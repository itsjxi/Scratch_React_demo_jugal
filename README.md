## Features
- Drag & drop block-based programming
- Multiple sprites with collision detection
- Data-driven architecture for easy extensibility
- Real-time animation execution

## Development

To run locally:

- Run `npm i` to install dependencies
- Run `npm start` and open http://localhost:3000 to see the app

## Architecture

This project uses a modern data-driven architecture where blocks are defined in configuration files rather than hardcoded components. See [ARCHITECTURE.md](./ARCHITECTURE.md) for details.
